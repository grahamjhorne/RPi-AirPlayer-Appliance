TWO WORKFLOWS EXPLAINED
═══════════════════════

WORKFLOW 1: Fresh Installation
  When: First time, new Pi, want clean state
  Time: ~17 minutes
  Steps: Image SD → Copy files → Run setup.sh → Reboot
  Result: Fresh, hardened Air Player appliance
  Docs: SETUP_GUIDE.md → Fresh Installation Workflow

WORKFLOW 2: Configuration Update (NEW v1.1)
  When: Change settings on running system
  Time: <1 minute + reboot if needed
  Steps: Edit properties → --dry-run → Run setup.sh → Reboot if needed
  Result: Updated configuration, license preserved
  Docs: SETUP_GUIDE.md → Configuration Update Workflow
